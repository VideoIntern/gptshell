# GPTSHELL #  

A python-based OpenAI client for terminal.

## Usage  

### Step 1. ###

 `pip install openai`

### Step 2. ###

Paste your SK on the **openai_api_key_store** variable to avoid duplicated initialization.

### Step 3. ###

Proxy on

### Step 4. ###

`python gptshell.py` 

## Note  

Remember to use ctrl+backspace to replace backspace.
